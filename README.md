# digital-scrapyard

A dumping ground for experimental react components

### > Requirements:

* Node
* Optional: Docker with docker compose
* Optional: GNU/Linux

### > Setup:

```
Without Docker:
1.) yarn install
2.) yarn serve
```

```
With Docker on GNU/Linux:
1.) ./project start
```

### > Why:

* To quickly develop my ideas
* To have a structured collection of components somewhere
* To have fun
